Easily change your wordpress body background color

== Description ==

Easy BG is a plugin that enables you to change the body color of your Wordpress site without the need to edit the code.  


== Installation ==

**Plugin Directory** 

1. Seach the plugin directory for plugin.
2. Click 'Install Now'.
3. Activate the plugin.
4. 'Easy BG' should be added to the side admin menu with the default settings.
5. You can modify the settings as needed.

**Upload** 

1. Download the plugin Zip file.
2. Go to plugins in the Wordpress Admin area.
3. Select 'Upload Plugin'
4. Activate the plugin.
5. 'Easy BG' should be added to the side admin menu with the default settings.
6. You can modify the settings as needed.
